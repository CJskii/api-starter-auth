# TypeScript Node.js Project

A starter template for a Node.js project with TypeScript.

## Getting Started

1. Install dependencies:
   ```
   npm install
   ```

2. Build the project:
   ```
   npm run build
   ```

3. Run the project:
   ```
   npm start
   ```

4. Run in development mode:
   ```
   npm run dev
   ```

## Scripts

- `build`: Compiles TypeScript to JavaScript
- `start`: Runs the compiled JavaScript
- `dev`: Runs TypeScript directly with ts-node
- `clean`: Removes the dist folder
